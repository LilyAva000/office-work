### 环境依赖

<details>
<summary>Windows</summary>

**用法示例：**
```bash
cd server
uv venv .venv --python=3.10
.\venv\Scripts\activate
uv pip install -r requirements.txt
```
</details>


<details>
<summary>Linux</summary>

**用法示例：**
```bash
cd server
uv venv .venv --python=3.10
source .venv/bin/activate
uv pip install -r requirements.txt
```
</details>
